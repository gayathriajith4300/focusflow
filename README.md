#FocusFlow

FocusFlow is a productivity web application built using HTML, CSS, and JavaScript. It brings essential productivity tools together in one place, helping users stay organized and focused without switching between multiple applications.

## Features

* 📝 To-Do List – Add, manage, and track daily tasks
* 📒 Notes – Capture ideas and save important information
* ⏱️ Pomodoro Timer – Improve focus using structured work and break sessions
* 💾 Local Storage – Automatically saves user data in the browser
* 📱 Responsive Design – Optimized for both desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API

This was my first individual web development project. Through FocusFlow, I explored front-end development concepts such as DOM manipulation, responsive design, browser storage, and debugging while building a practical application.
